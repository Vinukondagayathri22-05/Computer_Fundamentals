# Cyber Security Basics

## Introduction

Cyber Security is the practice of protecting computers, networks, software, and data from unauthorized access, attacks, and damage.

As internet usage increases, cyber threats also increase. Cyber security helps keep personal and organizational information safe.

---

# What is Cyber Security?

## Definition

Cyber Security is the protection of computer systems, networks, and data from cyber attacks and unauthorized access.

## Objectives

* Protect data
* Protect systems
* Maintain privacy
* Ensure secure communication

# 1. Malware

## Definition

Malware (Malicious Software) is any software designed to harm, damage, or gain unauthorized access to a computer system.

## Examples

* Virus
* Worm
* Trojan Horse
* Spyware
* Ransomware

## Effects

* Data theft
* System damage
* Performance slowdown

---

# 2. Virus

## Definition

A Virus is a type of malware that attaches itself to files or programs and spreads when those files are executed.

## Characteristics

* Requires user action
* Infects files
* Replicates itself

## Effects

* Corrupts files
* Slows computer performance
* Causes system crashes

## Prevention

* Use antivirus software
* Avoid unknown downloads

---

# 3. Worm

## Definition

A Worm is a malware program that can spread automatically across networks without user action.

## Characteristics

* Self-replicating
* Spreads rapidly
* Consumes network resources

## Effects

* Network congestion
* System slowdown

## Prevention

* Update software regularly
* Use firewalls

---

# 4. Trojan Horse

## Definition

A Trojan Horse is malware disguised as legitimate software.

## Characteristics

* Appears useful
* Contains hidden malicious code
* Does not self-replicate

## Examples

* Fake software downloads
* Fake games
* Cracked applications

## Effects

* Data theft
* Unauthorized access

## Prevention

* Download software from trusted sources only

---

# Difference Between Virus, Worm, and Trojan

| Virus                | Worm                  | Trojan                       |
| -------------------- | --------------------- | ---------------------------- |
| Attaches to files    | Spreads independently | Disguised as useful software |
| Requires user action | No user action needed | Requires user installation   |
| Self-replicates      | Self-replicates       | Does not self-replicate      |

---

# 5. Phishing

## Definition

Phishing is a cyber attack where attackers trick users into revealing sensitive information such as passwords, bank details, or personal information.

## Methods

* Fake emails
* Fake websites
* Fake messages

## Example

A fake email claiming to be from a bank asking for login credentials.

## Prevention

* Verify website URLs
* Avoid clicking suspicious links
* Never share passwords

---

# 6. Firewall

## Definition

A Firewall is a security system that monitors and controls incoming and outgoing network traffic.

## Functions

* Blocks unauthorized access
* Filters network traffic
* Protects systems from attacks

## Types

### Hardware Firewall

Installed as a physical device.

### Software Firewall

Installed as software on a computer.

---

# 7. Antivirus

## Definition

Antivirus software detects, prevents, and removes malware from a computer.

## Functions

* Scan files
* Detect threats
* Remove malware
* Provide real-time protection

## Examples

* Windows Defender
* Quick Heal
* Avast
* Norton

## Importance

Protects computers from viruses and other malware.

---

# 8. Password Security

## Definition

Password Security refers to creating and managing strong passwords to protect accounts and systems.

## Characteristics of a Strong Password

✔ At least 8–12 characters

✔ Combination of:

* Uppercase letters
* Lowercase letters
* Numbers
* Special characters

### Example

```text
G@yathri2026#
```

## Good Practices

* Use unique passwords
* Change passwords periodically
* Enable Two-Factor Authentication (2FA)
* Never share passwords
