# Software Concepts

## Introduction

A computer consists of two main components:

1. Hardware
2. Software

Hardware refers to the physical parts of a computer, while software refers to the programs and instructions that tell the hardware what to do.

Without software, computer hardware cannot perform any useful task.

---

# What is Software?

## Definition

Software is a collection of programs, instructions, and data that enables a computer to perform specific tasks.

## Examples

* Windows
* Linux
* Microsoft Word
* Google Chrome
* VLC Media Player

---

# Hardware vs Software

| Hardware            | Software                  |
| ------------------- | ------------------------- |
| Physical components | Programs and instructions |
| Can be touched      | Cannot be touched         |
| Example: Keyboard   | Example: MS Word          |
| Manufactured        | Developed                 |

### Example

```text
Hardware → Keyboard
Software → Typing Program
```

Both are required for a computer to work.

---

# Types of Software

Software is mainly classified into:

```text
Software
│
├── System Software
├── Application Software
├── Utility Software
└── Firmware
```

---

# 1. System Software

## Definition

System Software is software that manages computer hardware and provides a platform for running application software.

## Functions

* Controls hardware
* Manages memory
* Manages files
* Runs applications

## Examples

* Windows
* Linux
* macOS
* Android

### Importance

Without system software, application software cannot run.

---

# Components of System Software

### Operating System (OS)

Controls and manages the entire computer system.

Examples:

* Windows
* Linux
* macOS

### Device Drivers

Allow the operating system to communicate with hardware devices.

Examples:

* Printer Driver
* Graphics Driver
* Audio Driver

### Language Translators

Convert programming languages into machine language.

Examples:

* Compiler
* Interpreter
* Assembler

---

# 2. Application Software

## Definition

Application Software is designed to perform specific tasks for users.

## Examples

### General Purpose Software

* Microsoft Word
* Microsoft Excel
* PowerPoint
* Google Chrome

### Special Purpose Software

* Hospital Management System
* Banking Software
* Payroll Software

## Applications

* Writing documents
* Browsing the internet
* Watching videos
* Managing business operations

---

# 3. Utility Software

## Definition

Utility Software is software designed to maintain, protect, and optimize computer performance.

## Functions

* Disk cleanup
* Antivirus protection
* Backup management
* File compression

## Examples

* Antivirus Software
* Disk Cleanup Tool
* WinRAR
* Backup Utilities

## Advantages

* Improves system performance
* Protects against threats
* Maintains storage efficiency

---

# 4. Firmware

## Definition

Firmware is a special type of software permanently stored in ROM or Flash Memory that controls hardware devices.

## Characteristics

* Stored permanently
* Starts when device powers on
* Controls basic hardware functions

## Examples

* BIOS
* Router Firmware
* Smart TV Firmware
* Printer Firmware

---

# BIOS (Basic Input Output System)

## Definition

BIOS is firmware stored on a motherboard that starts the computer and checks hardware components.

## Functions

* Checks hardware during startup
* Loads the operating system
* Initializes system devices

# System Software vs Application Software

| System Software         | Application Software       |
| ----------------------- | -------------------------- |
| Runs the computer       | Performs user tasks        |
| Essential for operation | Depends on System Software |
| Starts automatically    | Started by users           |
| Example: Windows        | Example: MS Word           |

---

# Utility Software vs Application Software

| Utility Software     | Application Software       |
| -------------------- | -------------------------- |
| Maintains system     | Performs user tasks        |
| Improves performance | Solves specific user needs |
| Example: Antivirus   | Example: Excel             |

---

# Importance of Software

Software helps:

* Operate computer hardware
* Perform user tasks
* Manage resources
* Protect data
* Improve productivity

Without software, a computer is simply an electronic machine with no instructions.

---

# Summary

* Software is a collection of programs and instructions.
* System Software manages hardware and system operations.
* Application Software performs user-specific tasks.
* Utility Software maintains and protects the system.
* Firmware controls hardware at a basic level.
* BIOS is an important type of firmware.
