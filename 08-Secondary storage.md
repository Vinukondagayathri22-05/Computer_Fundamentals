# Secondary Storage

## Introduction

Secondary Storage is used to store data and programs permanently in a computer. Unlike RAM, data stored in secondary storage remains available even after the computer is turned off.

Secondary storage provides large storage capacity at a lower cost compared to primary memory.

---

# What is Secondary Storage?

## Definition

Secondary Storage is non-volatile memory used for the permanent storage of data, programs, operating systems, and files.

## Characteristics

* Non-volatile
* Permanent storage
* Large capacity
* Lower cost than primary memory
* Slower than RAM

---

# Need for Secondary Storage

Primary memory (RAM) loses data when power is turned off.

Secondary storage is needed to:

* Store operating systems
* Save documents
* Store photos and videos
* Store software applications
* Backup important data

---

Common secondary storage devices include:

---

# 1. Hard Disk Drive (HDD)

## Definition

A Hard Disk Drive (HDD) is a magnetic storage device used for storing large amounts of data permanently.

## Features

* Large storage capacity
* Uses magnetic disks
* Cost-effective
* Mechanical moving parts

## Storage Capacity

* 500 GB
* 1 TB
* 2 TB
* 4 TB and more

## Advantages

* Low cost
* Large storage space

## Disadvantages

* Slower than SSD
* Can be damaged by physical shock

---

# 2. Solid State Drive (SSD)

## Definition

A Solid State Drive (SSD) is a storage device that uses flash memory to store data.

## Features

* No moving parts
* Very fast
* Lightweight
* Energy efficient

## Applications

* Laptops
* Gaming PCs
* Workstations

## Advantages

* Faster boot time
* Better performance
* Durable

## Disadvantages

* More expensive than HDD

---

# Difference Between HDD and SSD

| HDD                  | SSD               |
| -------------------- | ----------------- |
| Uses magnetic disks  | Uses flash memory |
| Slower               | Faster            |
| Cheaper              | More expensive    |
| Moving parts present | No moving parts   |
| More noise           | Silent operation  |

---

# 3. Pen Drive

## Definition

A Pen Drive is a portable flash storage device used to transfer and store data.

## Features

* Small size
* Portable
* USB connection
* Reusable

## Storage Capacity

* 8 GB
* 16 GB
* 32 GB
* 64 GB
* 128 GB and more

## Applications

* File transfer
* Data backup
* Portable storage

## Advantages

* Easy to carry
* Fast data transfer

## Disadvantages

* Easy to lose due to small size

---

# 4. Memory Card

## Definition

A Memory Card is a small storage device commonly used in mobile phones, cameras, and tablets.

## Types

### SD Card

Secure Digital Card

### MicroSD Card

Smaller version used in smartphones

## Applications

* Smartphones
* Digital cameras
* Tablets

## Advantages

* Compact
* Portable

## Disadvantages

* Small size makes it easy to misplace

---

# 5. CD (Compact Disc)

## Definition

A CD is an optical storage device that stores data using laser technology.

## Storage Capacity

```text
Approximately 700 MB
```

## Applications

* Music storage
* Software distribution
* Data backup

## Advantages

* Low cost
* Easy distribution

## Disadvantages

* Limited storage capacity

---

# 6. DVD (Digital Versatile Disc)

## Definition

A DVD is an optical storage device with higher capacity than a CD.

## Storage Capacity

```text
4.7 GB (Single Layer)
8.5 GB (Dual Layer)
```

## Applications

* Movies
* Software installation
* Data storage

## Advantages

* Higher capacity than CD

## Disadvantages

* Slower than modern storage devices

---

# 7. Blu-ray Disc

## Definition

A Blu-ray Disc is a high-capacity optical storage device designed for high-definition video and large data storage.

## Storage Capacity

```text
25 GB (Single Layer)
50 GB (Dual Layer)
```

## Applications

* HD Movies
* Video Games
* Large Data Backup

## Advantages

* Very high storage capacity

## Disadvantages

* Requires Blu-ray compatible drive
