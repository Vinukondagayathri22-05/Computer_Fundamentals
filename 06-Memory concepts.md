# Memory Concepts

## Introduction

Memory is one of the most important components of a computer. It is used to store data, instructions, and results so that the CPU can access and process them when needed.

Without memory, a computer cannot store or retrieve information.

---

# 1. Data vs Information

## Data

### Definition

Data refers to raw facts, figures, or symbols that have not yet been processed.

### Examples

* 85
* Gayathri
* 22/05/2003
* A, B, C

These values alone may not have a clear meaning.

---

## Information

### Definition

Information is processed and organized data that has meaning and can be used for decision-making.

### Example

Data:

```text
Maths = 85
Science = 90
English = 80
```

Information:

```text
Total Marks = 255
Average = 85
Result = Pass
```

---

## Difference Between Data and Information

| Data                 | Information    |
| -------------------- | -------------- |
| Raw facts            | Processed data |
| Unorganized          | Organized      |
| No meaning by itself | Meaningful     |
| Input                | Output         |

---

# 2. Memory Units

Computers store information in binary form (0s and 1s).

Memory capacity is measured using different units.

---

# 3. Bit

## Definition

A Bit (Binary Digit) is the smallest unit of data in a computer.

## Values

A bit can have only two values:

```text
0 or 1
```

## Examples

```text
0
1
```

---

# 4. Byte

## Definition

A Byte is a group of 8 bits.

### Formula

```text
1 Byte = 8 Bits
```

## Example

```text
01000001
```

This binary value represents the character:

```text
A
```

---

# 5. Kilobyte (KB)

## Definition

A Kilobyte is larger than a byte.

### Formula

```text
1 KB = 1024 Bytes
```

## Example

Small text documents are often measured in KB.

---

# 6. Megabyte (MB)

## Definition

A Megabyte is larger than a Kilobyte.

### Formula

```text
1 MB = 1024 KB
```

## Example

* Photos
* Small applications

---

# 7. Gigabyte (GB)

## Definition

A Gigabyte is larger than a Megabyte.

### Formula

```text
1 GB = 1024 MB
```

## Example

* Smartphone storage
* Laptop RAM
* Movies

Examples:

* 8 GB RAM
* 16 GB RAM
* 256 GB SSD

---

# 8. Terabyte (TB)

## Definition

A Terabyte is larger than a Gigabyte.

### Formula

```text
1 TB = 1024 GB
```

## Example

* Hard disks
* Servers
* Data centers

Examples:

* 1 TB HDD
* 2 TB SSD

---

# 9. Petabyte (PB)

## Definition

A Petabyte is larger than a Terabyte.

### Formula

```text
1 PB = 1024 TB
```

## Example

Used by:

* Large companies
* Cloud providers
* Scientific research centers

---

# Why Binary is Used?

Computers use electronic circuits.

A circuit can be:

```text
ON  = 1
OFF = 0
```

Therefore computers use the Binary Number System to store and process information.

---

# Importance of Memory

Memory helps to:

* Store data
* Store programs
* Store operating systems
* Store results
* Retrieve information quickly

Without memory, computers cannot function effectively.

---

# Summary

* Data is raw facts and figures.
* Information is processed data.
* Bit is the smallest unit of memory.
* Byte consists of 8 bits.
* Memory units increase as:

Bit → Byte → KB → MB → GB → TB → PB

* Computers store data using binary values (0 and 1).
