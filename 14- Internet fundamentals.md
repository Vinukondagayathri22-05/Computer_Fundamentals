# Internet Fundamentals

## Introduction

The Internet is a global network that connects millions of computers and devices worldwide. It allows users to communicate, share information, access websites, send emails, and use online services.

The Internet has become an essential part of education, business, research, and daily life.

---

# What is the Internet?

## Definition

The Internet is a worldwide network of interconnected computers that communicate and share information using standard communication protocols.

## Uses of the Internet

* Information Searching
* Communication
* Online Education
* Social Media
* Online Banking
* E-commerce
* Entertainment
* Cloud Storage

---

# 1. World Wide Web (WWW)

## Definition

The World Wide Web (WWW) is a collection of web pages and websites accessible through the Internet.

It was invented by Tim Berners-Lee.

## Features

* Contains web pages
* Uses hyperlinks
* Accessible through web browsers

### Example

When you visit a website, you are using the World Wide Web.

---

# 2. Website

## Definition

A Website is a collection of related web pages available on the Internet under a single domain name.

## Examples

* Educational websites
* Shopping websites
* News websites
* Social media websites

### Example

A college website contains pages such as:

* Home
* About
* Courses
* Contact

---

# 3. Web Browser

## Definition

A Web Browser is software used to access and view websites on the Internet.

## Examples

* Google Chrome
* Microsoft Edge
* Mozilla Firefox
* Safari

## Functions

* Open websites
* Download files
* Search information
* Display web pages

---

# 4. Search Engine

## Definition

A Search Engine is a service that helps users find information on the Internet.

## Examples

* Google
* Bing
* Yahoo

## Working

```text
User Query
     ↓
Search Engine
     ↓
Relevant Results
```

### Example

Searching:

"Bioinformatics courses"

returns related web pages.

---

# 5. URL (Uniform Resource Locator)

## Definition

A URL is the address of a web page on the Internet.

### Example

```text
https://www.example.com
```

## Components of a URL

```text
https://www.example.com

https  → Protocol
www    → Subdomain
example → Domain Name
.com   → Domain Extension
```

---

# 6. Domain Name

## Definition

A Domain Name is the human-readable name of a website.

### Examples

```text
google.com
wikipedia.org
openai.com
```

## Advantages

* Easy to remember
* Replaces complex IP addresses

---

# 7. Email (Electronic Mail)

## Definition

Email is a method of sending and receiving electronic messages through the Internet.

## Components of an Email Address

### Example

```text
student@gmail.com
```

```text
student → Username
gmail   → Mail Service Provider
.com    → Domain Extension
```

## Uses

* Communication
* Sending files
* Academic notifications
* Professional correspondence

## Advantages

* Fast communication
* Low cost
* Worldwide accessibility

---

# 8. Cloud Computing Basics

## Definition

Cloud Computing is the delivery of computing services such as storage, servers, software, and databases over the Internet.

## Examples

* Google Drive
* OneDrive
* Dropbox

## Advantages

* Access from anywhere
* Data backup
* Easy sharing
* Reduced hardware cost

## Applications

* Online storage
* Data analysis
* Software services
* Bioinformatics research
