# Operating System Basics

## Introduction

An Operating System (OS) is the most important system software in a computer. It acts as an interface between the user and the computer hardware.

Without an operating system, a computer cannot perform useful tasks.

Examples:

* Windows
* Linux
* macOS
* Android
* iOS

---

# What is an Operating System?

## Definition

An Operating System (OS) is system software that manages computer hardware, software resources, and provides services for computer programs.

### Simple Definition

The Operating System acts as a manager of the entire computer system.

---

# Why Do We Need an Operating System?

The operating system helps:

* Run applications
* Manage files
* Manage memory
* Control hardware devices
* Provide security
* Enable user interaction

Without an OS, users would have to communicate directly with hardware, which is extremely difficult.

---

# Functions of an Operating System

## 1. Process Management

### Definition

Process management controls the execution of programs running on a computer.

### Functions

* Creates processes
* Schedules processes
* Terminates processes

### Example

Running:

* Chrome
* VS Code
* Spotify

at the same time.

---

## 2. Memory Management

### Definition

Memory management controls the allocation and usage of RAM.

### Functions

* Allocates memory
* Frees memory
* Prevents memory conflicts

### Example

Managing RAM while multiple applications are running.

---

## 3. File Management

### Definition

File management organizes and controls files and folders.

### Functions

* Create files
* Delete files
* Rename files
* Organize folders

### Example

Managing documents in Windows Explorer.

---

## 4. Device Management

### Definition

Device management controls communication between hardware devices and the operating system.

### Examples

* Printer
* Keyboard
* Mouse
* Monitor

### Uses Device Drivers

Drivers help the OS communicate with hardware.

---

## 5. Security Management

### Functions

* User authentication
* Password protection
* Access control

### Example

Windows login password.

---

## 6. User Interface

### Definition

Provides a way for users to interact with the computer.

### Types

#### GUI (Graphical User Interface)

Examples:

* Windows
* macOS
* Android

#### CLI (Command Line Interface)

Examples:

* Linux Terminal
* Windows Command Prompt

---

# Types of Operating Systems

## 1. Batch Operating System

### Definition

Processes jobs in batches without user interaction.

### Applications

* Payroll processing
* Bank transactions

### Advantage

Efficient for repetitive tasks.

---

## 2. Time-Sharing Operating System

### Definition

Allows multiple users to share computer resources simultaneously.

### Examples

* UNIX
* Linux

### Advantage

Efficient resource utilization.

---

## 3. Multi-Tasking Operating System

### Definition

Allows multiple applications to run simultaneously.

### Examples

* Windows
* Linux
* macOS

### Example

Using:

* Browser
* Music Player
* VS Code

at the same time.

---

## 4. Multi-User Operating System

### Definition

Allows multiple users to access the same computer system.

### Examples

* UNIX
* Linux Servers

---

## 5. Real-Time Operating System (RTOS)

### Definition

Provides immediate response to inputs and events.

### Applications

* Medical devices
* Air traffic control
* Robotics
