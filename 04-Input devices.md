# Input Devices

## Introduction

Input devices are hardware components used to enter data, instructions, and commands into a computer. They act as a bridge between the user and the computer system.

Without input devices, a computer cannot receive data from users.

---

# What is an Input Device?

## Definition

An input device is a hardware device that allows users to send data and instructions to a computer for processing.

## Examples

* Keyboard
* Mouse
* Scanner
* Microphone
* Webcam
* Barcode Reader
* Touch Screen

---

# 1. Keyboard

## Definition

A keyboard is the most commonly used input device used to enter text, numbers, and commands into a computer.

## Main Parts of a Keyboard

### Alphanumeric Keys

* A-Z
* 0-9

### Function Keys

* F1 to F12

### Control Keys

* Ctrl
* Alt
* Shift

### Navigation Keys

* Arrow Keys
* Home
* End
* Page Up
* Page Down

### Numeric Keypad

* Numbers and arithmetic operators

## Advantages

* Fast text entry
* Easy to use
* Essential for programming

## Disadvantages

* Requires typing skills

---

# 2. Mouse

## Definition

A mouse is a pointing device used to control the movement of the cursor on the screen.

## Types of Mouse

### Wired Mouse

Connected using a cable.

### Wireless Mouse

Connected through Bluetooth or USB receiver.

## Main Operations

### Click

Select an item.

### Double Click

Open a file or program.

### Right Click

Display shortcut menu.

### Drag and Drop

Move objects.

### Scroll

Move through pages.

## Advantages

* Easy navigation
* User-friendly

## Disadvantages

* Requires a flat surface

---

# 3. Scanner

## Definition

A scanner is an input device that converts printed documents, photos, and images into digital form.

## Working

1. Place document on scanner.
2. Scanner captures image.
3. Computer stores it as a digital file.

## Applications

* Digitizing documents
* Storing photographs
* OCR (Optical Character Recognition)

## Advantages

* Preserves physical documents digitally

## Disadvantages

* Large files may require more storage

---

# 4. Microphone

## Definition

A microphone is an input device used to capture sound and convert it into digital signals.

## Applications

* Voice recording
* Online meetings
* Speech recognition
* Voice assistants

## Advantages

* Easy audio input
* Supports communication

## Disadvantages

* Background noise may affect quality

---

# 5. Webcam

## Definition

A webcam is a camera connected to a computer that captures images and videos.

## Applications

* Video conferencing
* Online classes
* Live streaming
* Security monitoring

## Advantages

* Real-time video communication

## Disadvantages

* Image quality depends on camera resolution

---

# 6. Barcode Reader

## Definition

A barcode reader is an input device used to read barcodes and convert them into digital data.

## Working

1. Scanner reads barcode.
2. Barcode is decoded.
3. Information is sent to computer.

## Applications

* Supermarkets
* Libraries
* Inventory management
* Hospitals

## Advantages

* Fast data entry
* Reduces human errors

## Disadvantages

* Damaged barcodes may not be readable

---

# 7. Touch Screen

## Definition

A touch screen is both an input and output device that allows users to interact directly with the display using fingers or a stylus.

## Types

### Resistive Touch Screen

Responds to pressure.

### Capacitive Touch Screen

Responds to electrical touch from fingers.

## Applications

* Smartphones
* Tablets
* ATMs
* Information kiosks

## Advantages

* Easy to use
* No separate mouse or keyboard needed

## Disadvantages

* Fingerprints on screen
* May be less accurate for detailed work

---

# Importance of Input Devices

Input devices are important because they:

* Allow users to communicate with computers
* Enter data and instructions
* Improve efficiency
* Enable interaction with software applications

Without input devices, a computer cannot receive information from users.

---

# Summary

* Input devices are used to enter data into a computer.
* Keyboard is used for typing.
* Mouse is used for navigation.
* Scanner converts physical documents into digital form.
* Microphone captures audio.
* Webcam captures images and videos.
* Barcode Reader reads barcodes.
* Touch Screen allows direct interaction with the display.
